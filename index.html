PHL API

A collection of open data APIs from the City of Philadelphia. These APIs are being made available to participants in civic hackathons and other events to provide easy access to useful civic data that can be used to build powerful new software applications.

Crime Data API

Get access to the latest crime incident data, updated daily, going back to January 1, 2006. This data is available as a CSV download or through a powerful set of GeoAPIs that you can use directly from your software product or web application.

View more details »

L&I Data API (Beta)

Get access to a rich trove of city data on permits, licenses, cases and violation details from the City's department of Licenses and Inspections. This data is available through a simple yet powerful API built to conform to the OData specification.

View more details »

Open Data++

The City of Philadelphia makes dozens of open data sets available for developers, entrepreneurs and others to use at no charge. To find more open data sets and APIs that you can use to build civic software, check out the City's open data portal.
