Property Data API [BETA]

  • API endpoint
  • Properties by Address
  • Properties by Block
  • Nearby Properties
  • Properties by Intersection
  • Property by Account Number
  • Property by Property ID

This API exposes data on assessed property values based on information from the Office of Property Assessment (OPA). Currently in beta, subject to change.

API Endpoint

http://api.phila.gov/opa/v1.1/

Common Parameters

Name Type Default Description
format string xml Format of response (xml or json)
sort string full_address Field on which to sort results (ex. market_value)
dir string asc Direction of sorting (asc or desc)
limit number N/A Number of results returned
skip number N/A Number of results to skip (for pagination)

Search by Address

/address/<address>/<unit>
Searches properties by address string and, optionally, the unit. Returns an array of properties. Uses the City’s address validator to parse various address formats (ie. N or North and st or Street will all work). Note that the trailing slash after the address is still required if no unit is provided.
