This API exposes data on assessed property values based on information from the Office of Property Assessment. Currently in beta, subject to change.

API endpoint

The base URL for this API is http://api.phila.gov/OPA/v1.0/. Response formats supported are XML (by default), JSON (when a format=json parameter is passed) and JSONP (when a callback parameter is passed).
