311 API (Alpha)

This API exposes non-emergency service requests submitted to the City of Philadelphia through the Philly311 mobile application. It largely conforms to the Open311 (GeoReport V2) specification, with a few exceptions.

The Philly 311 API is currently read only - we only imlement API methods that allow a client to retrieve information about service types or service requests.

API Endpoints

Developmenthttp://www.publicstuff.com/api/open311/discovery.xml
Productionhttp://test.publicstuff.com/api/open311/discovery.xml

Get Services List

Get a list of 311 service request types and their associated service codes.

Sample URL:http://www.publicstuff.com/api/open311/services.json?&jurisdiction_id=philadelphia-pa
Response formats:XML, JSON
Sample Response:

  "service_code": 4152,

Get Service Definition

Get details associated with a service code. (This call is only necessary if the Service selected has metadata set as 'true' from the Get Services List response.)

Sample URL:http://www.publicstuff.com/api/open311/services/4086.json?&jurisdiction_id=philadelphia-pa
